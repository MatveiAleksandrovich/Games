## Ping-pong
Ping pong for two players, I hope it will help you have a good time with your friends.
### Technology
- Python
- PyGame
### Starting project
1. Clone repository `git clone https://github.com/MatveiAleksandrovich/Games.git`
2. Open `pong_game.py` in code editor.
3. Run `pong_game.py`
4. Have fun!
